## 1. 아이펠(AIFFEL)이란?

' 프랑스 ' 하면 가장 먼저 연상되는 **에펠탑(EIFFEL)** 처럼 누구나 함께 지식을 공유하고 함께 성장하는 

'교육 혁신의 상징'이 되고자 (주)**모두의연구소** 에서 설립한 자기 주도형 교육 플랫폼, **AI 혁신 학교** 입니다.
